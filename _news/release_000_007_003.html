Highlights

  • New Plunkr Template for Clarity

    With 0.7.3, Clarity has added a Plunkr template to make it easier to test Clarity and submit bug reports.

Bug Fixes

  • Replaced the SVG for the login background to use inline styles instead of the style tag to fix the IE 11 - Windows 10 issue with login backgrounds
  • Fixed horizontal scroll issues when website is viewed on small devices
  • Fixed a bug where the placeholder image and projected text were not displayed if the data was loading in the datagrid.
  • Fixed the compilation error in the datagrid test file
  • Fixed momentum scrolling issue when website was viewed on iPad
  • Removed Dropdown Menu close transition to eliminate the delay when the dropdown menu was closed

Other Changes

  • Added a dropdown with text toggle example in the header documentation
  • Updated the documentation to use the new icon markup for caret orientation
  • Reduced the distance of the dropdown caret from the text/image
