Getting Started

For who is this guide?

This guide is for everyone that want and needs to adopt our new Core components into there existing Clarity Angular application. It's design to help you rework and adopt in the best way and still get the same result as before.

How to use this guide?

The guide is structure into sections, every section will provide examples based on our Angular components and there Core counterparts. Into the form of guides, snippets and corner cases that will help you with the adoption of the new components and workflows.

Where to find more information?

The guide won't always reflect the latest API changes inside the components so if you want to know more on specific section you could go and check the official documentation
