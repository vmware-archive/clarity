@cds/core Grid with detail

Dev notes for grids with detail

Detail pane can be implemented by adding an action="button" to corresponding cds-grid-column and cds-grid-cell pairs. In side the cell, for each row you must use te cds-action-expand button. Bind the aria-label to details taht describe the row data. Listen to the click event and use that to toggle the visibility of the cds-grid-detail element. There are many ways to achieve this. This example uses *ngIf. Note, the cds-grid-detail element needs to bind to the [anchor] attribute. I passed it to the toggleDetail function so I could update the property and also pass the row details for the selected row.
