@cds/core multi-action grid

Dev notes for multi action grids

A multi action grid is a variant of the multi select grid. The only difference is that in this grid there is a cds-action button in the Host column. This action button toggles a popover that contains buttons to call functions that can act on the selected row objects. This is set up like other popovers.
