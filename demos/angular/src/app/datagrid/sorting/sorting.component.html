@cds/core Grid with column sorting

Dev notes for sortable grids

Sorting can be added to a grid with two changes. First, add a cds-action-sort button to the column that needs to be sorted. Then listen to the sortChange event on that button and implement the applications sorting algorithm, so that is takes the original data and places it on a sorted property that can be used as the iterator in the template for each of the rows.
