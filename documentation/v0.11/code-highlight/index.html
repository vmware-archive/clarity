Code Highlight

  • Examples & Code
  • Design Guidelines
Code Highlight is a custom directive to highlight code blocks using Prism.

Highlighting languages other than HTML, CSS, and JavaScript requires prism.css and prism.js to be included in the page.

<head>
    ...
    <link rel="stylesheet" href="node_modules/prismjs/themes/prism-solarizedlight.css">
    <script src="node_modules/prismjs/prism.js"></script>
    <script src="node_modules/prismjs/components/prism-typescript.min.js"></script>
    ...
</head>

To highlight a code block, add the clr-code-highlight directive to the code tag with the value set to the language-* style classes from PrismJS. Refer to the Prism documentation for the list of supported languages.

Example
<!-- "&#123;" left curly brace / "&#125;" right curly brace -->
<pre>
    <code clr-code-highlight="language-css">
        .some-component &#123;
            display: flex;
            flex-direction: column;
            justify-content: space-between;
            align-items: center;
            flex-wrap: wrap;
        &#125;
    </code>
</pre>
