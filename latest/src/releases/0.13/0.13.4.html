Highlights

  • Inline radio buttons

    Improved Angular radio button controls to support inline radios. Use the clrInline on the container to enable inline support. (#2693)

  • Datepicker new layout

    The datepicker now supports the new forms layout options, it will automatically change to the new layout if you have the new forms module in use, and it has a manual override if you need to set the layout type directly by using clrNewLayout. (#2631)
