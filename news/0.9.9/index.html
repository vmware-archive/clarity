Highlights

  • Clarity Signposts

    You can now use Clarity signposts clr-signpost to provide contextual info or help in a popover.

    
<clr-signpost>
    <clr-signpost-content *clrIfOpen>
        Lorem ipsum...
    </clr-signpost-content>
</clr-signpost>

    Read more on the Signpost docs page here.

  • Sketch Template Updated
    The Clarity sketch template has been updated with the latest changes in 0.9.9. This change includes the new color palette and fixes to forms, headers, wizard etc.

Bug Fixes

  • Fixed HideableColumnService error when using undefined column (non-hideable columns) in getColumnById.
  • Fixed issue with Datagrid Pagination emitting the default page value OnInit
  • Fixed an issue with the size of large spinners

Other Changes

  • Updated toggle design to better differentiate between off state and disabled states. 
    
<div class="toggle-switch">
    <input type="checkbox" id="checkbox1">
    <label for="checkbox1">Lorem ipsum Test</label>
</div>
  • Updated the modal max-height
