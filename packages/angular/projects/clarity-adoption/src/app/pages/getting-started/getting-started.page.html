Getting Started

For who is this guide?

This guide is for everyone that want and needs to adopt our new Core components into there existing Clarity Angular application. It's designt to help you rework and adopt in the best way and still get the same result as before.

How to use this guide?

The guide is structure into sections, every section will try to provide examples based on our Angular components and there Core counterparts. Inside there will be guides, snippets and corner cases that will help you with the adoption of the new components and workflows.

Eslint plugin

To help you with the adoption of core components we develop an eslint plugin that could pinpoint the places that you could update. To do that there is a plugin that must be instaled and configure.
