Summary of Options

[clrStepper] directive

Input / Output Values Default Effect
[clrInitialStep]
Type: String
Default: ''
 String '' Expects the name of the starting formGroupName or ngModelGroup to initialize stepper.

<clr-stepper-panel> component

Directive Values Default Effect
[formGroupName | ngModelGroup]
Type: FormGroupName | NgModelGroup
Default: ''
 FormGroupName | NgModelGroup '' Expects a formGroupName or ngModelGroup directive to initialize the stepper panel.

[clrStepButton] directive

Input / Output Values Default Effect
[clrStepButton]
Type: ClrStepButtonType | string
Default: ClrStepButtonType.Next | 'next'
 ClrStepButtonType | string ClrStepButtonType.Next | 'next' Expects a string value of 'next' or 'submit' to determine if button should navigate to next step or submit form.

*clrIfExpanded directive

Input/Output Values Default Effect
[(clrIfExpanded)]
Type: boolean
Default: false
 Boolean false Two-way binding on the expanded/collapsed state of the node.
