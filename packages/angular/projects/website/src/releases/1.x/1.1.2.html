Highlights

  • This is a minor version release that includes incremental changes and bug fixes to Clarity.
  • New Icons (#3192)

    One new shape was added, host-group (Technology). Two shapes were updated, dot-circle (Essential) circle (Essential).

Bug Fixes

  • Bugfix for tree view where it could trigger a flickering state by double clicking fast, or in any case where the tree tried to update before the tree had stabilized. (#3227).
  • Fixed caching issue with tree view component. (#3211).
  • Improved accessibility for alert component. (#3196).
  • Fixed bug causing incorrect form submit events when datagrid was nested in a form element. (#3190).
