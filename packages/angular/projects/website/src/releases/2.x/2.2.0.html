Highlights

  • Dynamic Popover Utility

    Clarity added a popover that can detect viewport clipping and eliminates issues with clipping of popovers in relatively positioned components. The ClrDatagrid is the first component to implement the utility for its internal popovers (filters, row actions and hideable columns).(#3720).

  • New Timeline component

    A static timeline component using HTML/CSS, based on the design spec from Clarity team (#3594)

  • New clrFocusOnViewInit Directive

    When using clrFocusOnViewInit on signpost content the screenreader reads the signpost content when it opens. (#3725)

  • New @Input for CLrDatepicker

    Users can now set a specific position for a datepicker.(#3731)

  • Inline Wizard Deprecation

    Now that Stepper exists, the need for an inline wizard (which is undocumented and psuedo-supported) is no longer needed or recommended. The inline wizard is now deprecated and will be removed in Clarity v4.0
