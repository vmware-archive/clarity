Breaking Changes

  • Sticky row actions in Datagrid (#3926)

    This is a visually breaking change. Row controls for actions like master/detail or select row have been moved into a sticky column and will remain visible when columns scroll horizontally.

  • Updated Color Palette (#3995)

    The Clarity color palette has been updated to use adjusted hsl values. This is a visually breaking change that will affect hues in the palette that are used in Clarity components. You can view the new color palettes here.

Highlights

  • Volume 16 Icons

    Volume 16 icons include: more social icons and technology icons.

  • Vertical Nav Accessibility Improvements

    Added aria-expanded attributes on the vertical nav element and on the vertical-nav-group elements to indicate expanded/collapsed state to screen readers (#4019)

Bug Fixes

  • Datagrid: Fixed clrDgFilterOpenChange not firing when open is toggled between true / false (#4000)
  • Datepicker: Prevent date input from displaying -1 day for specific timezones on mobile devices (#3993)
  • Datepicker: Prevent date from being deleted when selected on mobile (#3992)

Other Changes

  • Added 3.0 SASS rem-sizing variables (#3966)
  • Updated contribution guideline
