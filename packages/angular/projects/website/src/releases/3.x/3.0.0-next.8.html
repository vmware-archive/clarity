Breaking Changes

  • Update to Angular 9 (#4232)

    This release updates to the Angular v9.0.0. Note: The TypeScript version changed between Angular 8 and Angular 9 in a way that made Clarity unable to support both for Clarity 3. The TypeScript version to use with Clarity 3 is: v3.7.5

Highlights

  • Datagrid: Add API for disabling direct page access in pagination component (#4253)

    Declaring the [clrDgPageInputDisabled]="'true'" input onto the clr-dg-pagination component allows an application to disable user access to the pagination controls.

Bug Fixes

  • Updates for the clr-core icons API usage in alertys and badges (#4237)
  • Adjustment to Clarity action blue color for a11y(#4248)
