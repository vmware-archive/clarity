Highlights

Clarity moves forward with a new prerelease for the upcoming Clarity v4 milestone. There are a number of breaking changes and improvements for this version, which will only work with Angular 10 and TypeScript 3.9.

  • Support Datagrid column resizing with keyboard for accessibility (#4535)

    Support Datagrid expandable rows for accessibility (#4163)

    Success helper message in forms (#4565)

    Form input group support (#4403)

    Update to Angular 10 (#4556)

    Update several colors for accessibility (#4707)

    Fix z-index so column switcher is above modals (#4693)

    Fix possible undefined error when opening a popover synchronously (#4470)

    Fix selection for server-driven Datagrids in some uses (#4645)

    Alert Fix alert text width in IE11 (#4557)

    Accessibility Remove internal use of ClrAriaLiveService (#4687)
