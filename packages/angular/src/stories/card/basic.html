All About Cards
A card presents high-level information and can guide the user toward related actions and details. Adding the .clickable class makes the entire card clickable, initiating a single action. The footer can contain two actions. For more actions, use a dropdown. Cards can also contain alerts and lists and groups and progress indicators. See the additional stories for more examples.
