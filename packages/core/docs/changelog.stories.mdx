# Changelog

## 4.0.0

4.0 introduces many new components including enhanced alerts, modals and form controls.
React is now officially supported with our new `@clr/react` package.
Several bugfixes have been made to improve performance and treeshaking of components.

### React Support

React is now officially supported with our new `@clr/react` package. Follow our
[React Documentation](./?path=/story/documentation-react--page) to get started.

```bash
npm install @clr/react
```

```jsx
import { CdsAlert } from '@clr/react/alert';

<CdsAlert status="info" closable={this.state.closable} onCloseChange={this.log}>
  Hello World
</CdsAlert>;
```

### Breaking Changes

- ### Import Paths
  To future proof performance and support more advanced use cases the import statement
  for components has changed. Imports for components now end with `register` path.

```javascript
import '@cds/core/button'; // 3.0

import '@cds/core/button/register'; // 4.0+
```

- ### Alerts / Alert Groups

  The Alert APIs are revamped to support new features including alert groups/actions as
  well as lightweight alerts which are now exclusively in Core. Alerts APIs will
  remain stable in 4.0+

  - [Lightweight Alerts](./?path=/story/components-alert-getting-started--page)
  - [Alert Groups](./?path=/story/components-alert-group-getting-started--page)
