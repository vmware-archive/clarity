# Accordion

An accordion allows generic content to be collapsed and allows users to expand to show more detail.

An accordion is a container for one or more accordion panels.

## Installation

To use the accordion component import the component in your JavaScript.

```typescript
import '@clr/core/accordion/register.js';
```

```html
<cds-accordion>
  <cds-accordion-panel>
    <cds-accordion-header>accordion header</cds-accordion-header>
    <cds-accordion-content>accordion content</cds-accordion-content>
  </cds-accordion-panel>
</cds-accordion>
```
