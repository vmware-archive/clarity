# Accordion

An accordion allows generic content to be collapsed and allows users to expand to show more detail.

An accordion is a container for one or more accordion panels.

## Installation

To use the accordion component import the component in your JavaScript.

```typescript
import '@cds/core/accordion/register.js';
```

```html
<cds-accordion>
  <cds-accordion-panel expanded>
    <cds-accordion-header>Item 1</cds-accordion-header>
    <cds-accordion-content>Content 1</cds-accordion-content>
  </cds-accordion-panel>

  <cds-accordion-panel>
    <cds-accordion-header>Item 2</cds-accordion-header>
    <cds-accordion-content>Content 2</cds-accordion-content>
  </cds-accordion-panel>

  <cds-accordion-panel disabled>
    <cds-accordion-header>Item 3</cds-accordion-header>
    <cds-accordion-content>Content 3</cds-accordion-content>
  </cds-accordion-panel>
</cds-accordion>
```
