# Badge

Badges provide a method to highlight a count of an element either next to it
or inside the element itself.

## Installation

To use the badge component import the component in your JavaScript.

```typescript
import '@cds/core/badge/register.js';
```

```html
<cds-badge status="info">
  2
  <span cds-layout="display:screen-reader-only"
    >items. Item text for screen-readers should be added to badges in elements that will only be read in a
    screen-reader.</span
  >
</cds-badge>
```
