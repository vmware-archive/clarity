# Breadcrumbs

Breadcrumbs are a type of secondary navigation scheme that reveals the userâ€™s location in a site or web app.

## Installation

To use the breadcrumb component import the component in your JavaScript.

```typescript
import '@cds/core/breadcrumb/register.js';
```

```html
<cds-breadcrumb>
  <a href="#">Home</a>
  <a href="#">Parent page</a>
  <span>Current page</span>
</cds-breadcrumb>
```
