Action buttons are icon style buttons with defined behaviors. The standard `cds-button-action`
provides the proper touch target and states for icon buttons. The `cds-button-expand` is for actions that
trigger content to be expanded or shown to the user. The `cds-button-expand` provides three
expand action styles `'vertical' | 'horizontal' | 'detail'`.

## Installation

```typescript
import '@cds/core/button-expand/register.js';
```

## Usage

```html
<cds-button-expand expanded action="vertical" aria-label="close details"></cds-button-expand>
<cds-button-expand action="horizontal" aria-label="show more"></cds-button-expand>
<cds-button-expand action="detail" aria-label="show details"></cds-button-expand>
```
