Action buttons are icon style buttons with defined behaviors. The standard `cds-button-action` button
provides the proper touch target and states for icon buttons. The `cds-button-sort` provides the
proper accessible aria attributes for sortable type features. To defined the current sort state
set the `sort` attribute/property to one of the following `ascending/descending/none`.

## Installation

```typescript
import '@cds/core/button-sort/register.js';
```

## Usage

```html
<cds-button-sort sort="none" aria-label="sort item"></cds-button-sort>
<cds-button-sort sort="ascending" aria-label="sort item"></cds-button-sort>
<cds-button-sort sort="descending" aria-label="sort item"></cds-button-sort>
```
