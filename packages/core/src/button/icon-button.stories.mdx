# Icon Buttons

Icon buttons allow an application to communicate action and direct user intent in a compact and visually appealing manner.

## Installation

To use the icon button component import the component in your JavaScript, as well as the icon you want to use. Note that your application needs to have imported both the cds-icon component as well as any icon shape you want to use.

```typescript
import '@cds/core/icon/register.js';
import '@cds/core/button/register.js';
import { ClarityIcons, downloadIcon } from '@cds/core/icon';

ClarityIcons.add(downloadIcon);
```

```html
<cds-icon-button><cds-icon shape="download"></cds-icon></cds-icon-button>
```
