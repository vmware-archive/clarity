# Inline Buttons

Inline buttons are used inside and alongside textual content within Clarity components. They give action buttons a less prominent, yet familiar, visual presence.

## Installation

To use the inline button component import the component in your JavaScript.

```typescript
import '@cds/core/button/register.js';
```

```html
<cds-inline-button>Click Here</cds-inline-button>
```
