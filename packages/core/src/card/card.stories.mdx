# Card

## Installation

```typescript
import '@cds/core/card/register.js';
```

## Usage

```html
<cds-card>
  <div cds-layout="vertical gap:md" aria-labelledby="{id}">
    <h2 id="{id}" cds-text="section" cds-layout="p-y:sm">Card Title</h2>

    <cds-divider cds-card-remove-margin></cds-divider>

    <div cds-text="body light" cds-layout="p-y:md">Card Content</div>

    <cds-divider cds-card-remove-margin></cds-divider>

    <div cds-layout="horizontal gap:sm p-y:sm align:vertical-center">
      <cds-button action="flat-inline">View</cds-button>
    </div>
  </div>
</cds-card>
```
