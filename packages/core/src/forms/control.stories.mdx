# Generic Control

The Control provides a way to leverage the same Clarity Form layout but with a custom input control.

## Installation

To use the control component import the component in your JavaScript.

```typescript
import '@cds/core/forms/register.js';
```

```html
<cds-control>
  <label>label text</label>
  <input placeholder="some custom control" />
  <cds-control-message>message text</cds-control-message>
</cds-control>
```
