Grid provides several alginment options for columns. By default content is algined to the left.

## Align Center

```html
<cds-grid column-align="center">
  <cds-grid-column>Host</cds-grid-column>
  <cds-grid-column>Status</cds-grid-column>
  <cds-grid-column>CPU</cds-grid-column>
  <cds-grid-column>Memory</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>vm-host-001</cds-grid-cell>
    <cds-grid-cell>online</cds-grid-cell>
    <cds-grid-cell>5%</cds-grid-cell>
    <cds-grid-cell>10%</cds-grid-cell>
  </cds-grid-row>
</cds-grid>
```

## Align Right

```html
<cds-grid column-align="right">
  <cds-grid-column>Host</cds-grid-column>
  <cds-grid-column>Status</cds-grid-column>
  <cds-grid-column>CPU</cds-grid-column>
  <cds-grid-column>Memory</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>vm-host-001</cds-grid-cell>
    <cds-grid-cell>online</cds-grid-cell>
    <cds-grid-cell><p cds-text="monospace">5.00%</p></cds-grid-cell>
    <cds-grid-cell><p cds-text="monospace">10.00%</p></cds-grid-cell>
  </cds-grid-row>
</cds-grid>
```
