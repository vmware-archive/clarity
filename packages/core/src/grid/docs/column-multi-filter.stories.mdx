To search any column in a Data Grid, use `cds-search` and the HTML `input` event
to filter your data set and render only matching rows.

<Story id="stories-grid--column-multi-filter" />

```html
<section cds-layout="vertical gap:lg">
  <cds-search>
    <label>Search VMs</label>
    <input type="search" placeholder="search" />
  </cds-search>
  <cds-grid>
    <cds-grid-column>Host</cds-grid-column>
    <cds-grid-column>Status</cds-grid-column>
    <cds-grid-column>CPU</cds-grid-column>
    <cds-grid-column>Memory</cds-grid-column>
    <cds-grid-row>
      <cds-grid-cell>vm-host-001</cds-grid-cell>
      <cds-grid-cell>online</cds-grid-cell>
      <cds-grid-cell>5%</cds-grid-cell>
      <cds-grid-cell>10%</cds-grid-cell>
    </cds-grid-row>
  </cds-grid>
</section>
```

Any strategy can be used to determine what cells to search or what weights
should apply. Here is an example of a basic string matching filter.

```javascript
const results = itemsList.filter(i =>
  Object.keys(i)
    .map(k => i[k])
    .reduce((p, n) => `${p} ${n}`)
    .toLocaleLowerCase()
    .includes(searchInputElement.value.trim().toLocaleLowerCase())
);
```
