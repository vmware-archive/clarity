
## Usage

The Core Data Grid functions much like an HTML Table Element. The Grid is stateless only rendering what HTML is provided.
Interactions are not managed by the Data Grid but rather the host application. This provides significantly more flexibility
for how the state is managed for your Data Grid.

```html
<cds-grid>
  <cds-grid-column>Host</cds-grid-column>
  <cds-grid-column>Status</cds-grid-column>
  <cds-grid-column>CPU</cds-grid-column>
  <cds-grid-column>Memory</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>vm-host-001</cds-grid-cell>
    <cds-grid-cell>online</cds-grid-cell>
    <cds-grid-cell>5%</cds-grid-cell>
    <cds-grid-cell>10%</cds-grid-cell>
  </cds-grid-row>
  <cds-grid-footer></cds-grid-footer>
</cds-grid>
```
