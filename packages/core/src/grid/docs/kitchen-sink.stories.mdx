# Data Grid Kitchen Sink

The kitchen sink demo integrates many of the grid features into a single demo. Because the Grid is stateless, the Grid can be
re-rendered from any state source. In this demo any updates are stored in local storage. Refresh the page, and the Grid
is re-rendered from that same state.
