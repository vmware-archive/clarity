# Grid Row Action

<cds-alert-group status="warning" cds-layout="m-b:lg">
  <cds-alert>
    This component or utility is offered as a preview. This means we are currently working on it and seeking feedback.
    Please be aware that this component or utility may have breaking changes before we finish working on it.
  </cds-alert>
</cds-alert-group>

Row Actions can be applied to `cds-grid-row` by leveraging the `cds-button-action`.
The `cds-button-action` can open a context dropdown per row to allow the user to
trigger an action specific to that row.

<Story id="stories-grid--row-action" />

<cds-alert status="info" cds-layout="m-t:lg">When providing actions to rows, ensure the action menu is not repeated for each row.
Reuse the same action menu for all rows to prevent unnecessary DOM work.</cds-alert>

```html
<cds-grid>
  <cds-grid-column type="action"></cds-grid-column>
  <cds-grid-column>Host</cds-grid-column>
  <cds-grid-column>Status</cds-grid-column>
  <cds-grid-column>CPU</cds-grid-column>
  <cds-grid-column>Memory</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>
      <cds-button-action aria-label="choose available host actions"></cds-button-action>
    </cds-grid-cell>
    <cds-grid-cell>vm-host-001</cds-grid-cell>
    <cds-grid-cell>online</cds-grid-cell>
    <cds-grid-cell>5%</cds-grid-cell>
    <cds-grid-cell>10%</cds-grid-cell>
  </cds-grid-row>
</cds-grid>
<cds-dropdown hidden>
  <cds-button block action="flat" size="sm">Buy APPL</cds-button>
  <cds-button block action="flat" size="sm">Sell APPL</cds-button>
</cds-dropdown>
```
