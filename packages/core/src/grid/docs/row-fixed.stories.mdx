# Grid Row Fixed

<cds-alert-group status="danger" cds-layout="m-b:lg">
  <cds-alert>
    This feature or component is <strong>experimental</strong> and has no defined support policy. Experimental features
    are subject to change at any time and are not stable.
  </cds-alert>
</cds-alert-group>

Rows can be fixed in place. This will prevent the row from scrolling out of view.
To place a row in a fixed position, set the column with a `position="fixed"` attribute. You can enable user toggling
of the position by using a `cds-action` to set the position state of a given column.

<cds-alert status="warning">Only the first row can be placed in the fixed position and cannot be sorted.</cds-alert>

  <cds-grid-row position="fixed">
