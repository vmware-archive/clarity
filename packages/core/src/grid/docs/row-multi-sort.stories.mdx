# Grid Row Multi Sort

Grid Columns can render a sort button component by setting the `sort` attribute/property.
The Column will emit a `sortChange` event when the user interacts with the Sort button.
The supported sort types are `none`, `ascending` and `descending`. Using the `sortChange`
event you can re-sort rows within the application.

<Story id="stories-grid--row-multi-sort" />

<br />

Multi-Sort can be accomplished by applying `sort` to multiple columns. Because the `sortChange`
event reports the sort type, only the host application can choose any kind of sort logic. The
order or weight of a sort is up to the host application. Conventionally, Column sorts apply
in order of the columns.

```javascript
document.querySelector('#host').addEventListener('sortChange', e => sortByString(e.detail));
document.querySelector('#status').addEventListener('sortChange', e => sortByNumber(e.detail));
```

```html
<cds-grid>
  <cds-grid-column> Host <cds-action-sort sort="decending" aria-label="sort host"></cds-action-sort> </cds-grid-column>
  <cds-grid-column>
    Status <cds-action-sort sort="accending" aria-label="sort status"></cds-action-sort>
  </cds-grid-column>
  <cds-grid-column>CPU</cds-grid-column>
  <cds-grid-column>Memory</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>vm-host-001</cds-grid-cell>
    <cds-grid-cell>online</cds-grid-cell>
    <cds-grid-cell>5%</cds-grid-cell>
    <cds-grid-cell>10%</cds-grid-cell>
  </cds-grid-row>
</cds-grid>
```
