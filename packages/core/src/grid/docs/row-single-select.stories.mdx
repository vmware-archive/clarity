# Grid Row Single Select

<cds-alert-group status="warning" cds-layout="m-b:lg">
  <cds-alert>
    This component or utility is offered as a preview. This means we are currently working on it and seeking feedback.

Single select rows are created using the `cds-radio` component. Create an associated
radio group by using the same `name` attribute on each radio control. To mark a row
as selected, set the `selected` attribute/property on the `cds-grid-row` component.

<Story id="stories-grid--row-single-select" />

```html
<cds-grid selectable="single">
  <cds-grid-column type="action"></cds-grid-column>
  <cds-grid-column>Host</cds-grid-column>
  <cds-grid-column>Status</cds-grid-column>
  <cds-grid-column>CPU</cds-grid-column>
  <cds-grid-column>Memory</cds-grid-column>
  <cds-grid-row selected>
    <cds-grid-cell>
      <cds-radio>
        <input type="radio" name="hosts" aria-label="select host vm-host-001" />
      </cds-radio>
    </cds-grid-cell>
    <cds-grid-cell>vm-host-001</cds-grid-cell>
    <cds-grid-cell>online</cds-grid-cell>
    <cds-grid-cell>5%</cds-grid-cell>
    <cds-grid-cell>10%</cds-grid-cell>
  </cds-grid-row>
</cds-grid>
```
