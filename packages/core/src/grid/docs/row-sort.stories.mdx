# Grid Row Sort

<cds-alert-group status="warning" cds-layout="m-b:lg">

Grid Columns can render a sort button component by setting the `sort` attribute/property.
The Action Sort will emit a `sortChange` event when the user interacts with the Sort button.
The supported sort types are `none`, `ascending` and `descending`. Using the `sortChange`
event you can re-sort rows within the application.

<Story id="stories-grid--row-sort" />

```html
<cds-grid>
  <cds-grid-column>Host <cds-action-sort sort="ascending" aria-label="sort"></cds-action-sort></cds-grid-column>
  <cds-grid-column>Status</cds-grid-column>
  <cds-grid-column>CPU</cds-grid-column>
  <cds-grid-column>Memory</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>vm-host-001</cds-grid-cell>
    <cds-grid-cell>online</cds-grid-cell>
    <cds-grid-cell>5%</cds-grid-cell>
    <cds-grid-cell>10%</cds-grid-cell>
  </cds-grid-row>
</cds-grid>
```
