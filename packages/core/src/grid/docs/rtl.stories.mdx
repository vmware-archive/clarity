Data Grid supports internationalization via the [I18nService](./?path=/story/foundation-internationalization--page). The Data Grid also supports
RTL languages. To enable RTL support, apply the `dir="rtl"` attribute to the Data Grid.

<Story id="stories-grid--rtl" />

```html
<cds-grid dir="rtl">
  <cds-grid-column>Stock</cds-grid-column>
  <cds-grid-column>Average</cds-grid-column>
  <cds-grid-column>Current</cds-grid-column>
  <cds-grid-column>About</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>APPL</cds-grid-cell>
    <cds-grid-cell>$1000</cds-grid-cell>
    <cds-grid-cell>$0</cds-grid-cell>
    <cds-grid-cell>...</cds-grid-cell>
  </cds-grid-row>
</cds-grid>
```
