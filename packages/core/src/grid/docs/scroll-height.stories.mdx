# Grid Scroll Height

<cds-alert-group status="warning" cds-layout="m-b:lg">

The `height` property allows control of the scroll height of the grid. By default, the grid will expand to the number of provided rows.

```html
<cds-grid aria-label="height datagrid demo" height="400">
  <cds-grid-column>Host</cds-grid-column>
  <cds-grid-column>Status</cds-grid-column>
  <cds-grid-column>CPU</cds-grid-column>
  <cds-grid-column>Memory</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>vm-host-001</cds-grid-cell>
    <cds-grid-cell>online</cds-grid-cell>
    <cds-grid-cell>5%</cds-grid-cell>
    <cds-grid-cell>10%</cds-grid-cell>
  </cds-grid-row>
  <cds-grid-footer></cds-grid-footer>
</cds-grid>
```

## Min Height

You can also set a minimum grid body height allowing the grid using the `--body-min-height` css custom property.

```html
<cds-grid aria-label="min height datagrid demo" style="--body-min-height: 400px">
  ...
</cds-grid>
```
