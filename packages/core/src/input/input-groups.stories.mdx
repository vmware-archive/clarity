# Input Groups and Actions

Text groups allow closely related input controls to be combined into a visually
single control.

## Installation

To use the input group component import the component in your JavaScript.

```typescript
import '@cds/core/input/register.js';
```

```html
<cds-input-group>
  <label>Host URL</label>
  <cds-select cds-layout="align:shrink">
    <label>URL Protocol</label>
    <select>
      <option>https://</option>
      <option>http://</option>
    </select>
  </cds-select>
  <cds-input>
    <label>Host URL</label>
    <input placeholder="example.com" type="url" />
  </cds-input>
  <cds-control-message>Host ID: 123456</cds-control-message>
</cds-input-group>
```
