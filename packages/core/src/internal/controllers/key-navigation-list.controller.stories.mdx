# Key Navigation - List

<cds-alert-group status="danger" cds-layout="m-b:lg">
  <cds-alert>This is an internal API and should only be used for developing Clarity Components.</cds-alert>
</cds-alert-group>

The key navigation controllers are generic key handlers that enable various keyboard navigation behavior.

The `@keyNavigationList` controller enables a component to support key navigation through a
one dimentional linear list of elements. The controller also provides shortcut keys such
as `Home`, `End`, `PageUp` and `PageDown`.

```typescript
@keyNavigationList<DemoKeyNavigationList>()
class DemoKeyNavigationList extends LitElement {
  @queryAll('section > div') keyListItems: NodeListOf<HTMLElement>;

  @state() private items = Array.from(Array(10).keys());

  render() {
    return html`
      <section @cdsKeyChange=${...}>
        ${this.items.map(i => html`<div><button>${i}</button></div>`)}
      </section>
    `;
  }
}
```
