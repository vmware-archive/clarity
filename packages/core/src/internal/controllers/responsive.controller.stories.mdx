# Responsive Controller

<cds-alert-group status="danger" cds-layout="m-b:lg">
  <cds-alert>This is an internal API and should only be used for developing Clarity Components.</cds-alert>
</cds-alert-group>

The Responsive Controller enables a given component to get updates when the component
dimentions are changed or resized. On resize the `cdsResizeChange` event will emit from
the host element providing a [`DOMRect`](https://developer.mozilla.org/en-US/docs/Web/API/DOMRect) instance.

```typescript
class ResponsiveControllerTestElement extends LitElement {
  responsiveController = new ResponsiveController(this);

  connectedCallback() {
    super.connectedCallback();
    this.addEventListener('cdsResizeChange', event => console.log(event)); // DOMRect
  }
}
```
