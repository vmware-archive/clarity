# Lists

Lists in Clarity Core come in three varieties â€“ ordered, unordered, and
unstyled. Any "compact" style variations for list display is now handled through

## Installation

To use the list styles import the global CSS.

```scss
// import into sass or css
@import '~@clr/core/list/list.min';

// or import in html
<link href="/node_modules/@clr/core/list/list.min.css" rel="stylesheet">
```
