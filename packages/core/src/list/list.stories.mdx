# Lists

Lists in Clarity Core come in four varieties â€“ ordered, unordered,
unstyled, and description lists. Any "compact" style variations for
list display is now handled through Clarity Core layouts.

## Installation

To use the list styles import the global CSS.

```scss
// import into sass or css
@import '~@cds/core/list/list.min';

// or import in html
<link href="/node_modules/@cds/core/list/list.min.css" rel="stylesheet">
```
