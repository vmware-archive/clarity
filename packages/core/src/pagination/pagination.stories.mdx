# Pagination

<cds-alert-group status="warning" cds-layout="m-b:lg">
  <cds-alert>

Pagination component is used when data items need to be presented into several pages, and only one page (it can contain one or more items) will be loaded at a time. The component must allow users to browse the pages.

## Installation

To use the Pagination component import the component in your JavaScript.

```typescript
import '@cds/core/pagination/register.js';
```

```html
<cds-pagination aria-label="pagination">
  <cds-pagination-button action="first" aria-label="go to first" disabled></cds-pagination-button>
  <cds-pagination-button action="prev" aria-label="go to previous" disabled></cds-pagination-button>
  <span aria-label="current page">1 / 3</span>
  <cds-pagination-button action="next" aria-label="go to next"></cds-pagination-button>
  <cds-pagination-button action="last" aria-label="go to last"></cds-pagination-button>
</cds-pagination>
```

To meet a11y standards, we strongly recommend using `aria-label` for each pagination elements.
