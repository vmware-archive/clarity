# Circular Progress Bar and Spinner

<cds-alert-group status="warning" cds-layout="m-b:lg">

The circular progress bar is a compact indicator for providing feedback about an ongoing, user-initiated process.

In its indeterminate state, the circular progress bar functions as a "spinner" which indicates an ongoing process of unknown duration.

## Installation

To use the circular progress component, import the component in your JavaScript.

```typescript
import '@cds/core/progress-circle/register.js';
```

```html
<cds-progress-circle status="info" value="25"></cds-progress-circle>
```
