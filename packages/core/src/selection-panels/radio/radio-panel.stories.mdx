# Complex Radio

Radio panels are buttons that allow users to select one option from a mutually exclusive group of options. It also
provides an increased capacity to display additional information related to each option.

## Installation

To use the checkbox panel, import the component in your JavaScript.

```typescript
import '@cds/core/selection-panels/radio/register.js';
```

```html
<cds-radio-group>
  <cds-radio-panel>
    <label cds-layout="vertical gap:md align:center">
      <span cds-text="section">VM One</span>
      <span cds-text="subsection center">Orchestrate & Automate</p>
    </label>
    <input type="radio" value="0" />
  </cds-radio-panel>
  <cds-radio-panel>
    <label cds-layout="vertical gap:md align:center">
      <span cds-text="section">VM One</span>
      <span cds-text="subsection center">Orchestrate & Automate</p>
    </label>
    <input type="radio" value="1" />
  </cds-radio-panel>
</cds-radio-group>
```
