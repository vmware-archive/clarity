## Installation

To use the color tokens import the global CSS.

```scss
// import into sass or css
@import '~@clr/core/styles/module.tokens.min';

// or import in html
<link href="/node_modules/@clr/core/styles/module.tokens.min.css" rel="stylesheet">
```

## Usage

```css
.my-element {
  background: var(--cds-token-color-action-1000);
  color: var(--cds-token-color-neutral-0);
}
```
