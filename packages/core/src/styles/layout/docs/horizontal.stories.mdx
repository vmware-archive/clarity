# Horizontal

<cds-alert status="warning" closable="false" cds-layout="m-b:lg">
  <cds-alert-content>
    This is an experimental API and likely will have breaking changes in the near future.
  </cds-alert-content>
</cds-alert>

The horizontal layout type enables horizontal rows of items that can wrap automatically.
Spacing, alignments and gap between items can all be configured with the component.
Gap values are t-shirt size `[xxs, xs, sm, md, lg, xl, xxl]` and align with the [Layout Spacing Tokens](./?path=/docs/experimental-spacing-getting-started--page).

## Horizontal Layout

Horizontal layout is useful for horizontal lists of items such as buttons,
tags or labels.

  <Story id="experimental-layout-stories--horizontal-layout" />
</Preview>

## Wrap

By default the horizontal layout will allow items to wrap and ensure the gap
is only applied between layout items.

<Preview>
  <Story id="experimental-layout-stories--horizontal-layout-wrap" />
</Preview>

## No Wrap

You can disable wrapping using the `wrap="none"` attribute. This will force
the items to follow standard CSS overflow rules (`hidden`/`scroll`).

  <Story id="experimental-layout-stories--horizontal-layout-no-wrap" />
</Preview>
