# Vertical

The vertical layout type enables stacks of items. This is useful for spacing
cards, content and other block level items. Spacing, alignments and gap between
stacked items can all be configured with the component. Gap values are t-shirt size `[xxs, xs, sm, md, lg, xl, xxl]`
and align with the [Layout Spacing Tokens](./?path=/docs/experimental-spacing-getting-started--page).
