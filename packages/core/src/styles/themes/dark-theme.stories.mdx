# Dark Theme

Clarity provides a built in dark theme which can be loaded and dynamically enabled
in the browser.

```scss
// import into sass or css
@import '~@cds/core/styles/theme.dark.min.css';

// or import in html
<link href="/node_modules/@cds/core/styles/theme.dark.min.css" rel="stylesheet">
```

```html
<body cds-theme="dark">
  ...
</body>
```
