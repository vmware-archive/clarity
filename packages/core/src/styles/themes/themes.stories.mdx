# Themes

button above. To use the dark theme, import the global CSS file.

```scss
// import into sass or css
@import '~@cds/core/styles/theme.dark.min.css';

// or import in html
<link href="/node_modules/@cds/core/styles/theme.dark.min.css" rel="stylesheet">
```

Then on the body element add the following attribute.

```html
<body cds-theme="dark">
  ...
</body>
```
