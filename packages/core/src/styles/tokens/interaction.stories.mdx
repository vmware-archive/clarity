# Interaction Styles

UI objects which are interactive should use the `--cds-alias-object-interaction-*` tokens.
The object interaction tokens define the look and feel of interactive elements within
Clarity. All interactions consist of the following five interaction states.

- Hover
- Active
- Selected
- Focus
- Disabled

These tokens do not apply to interactive button elements such as the primary and
outline buttons. Below are example demos with various interaction patterns using
the interaction tokens. The complete list of interaction tokens can be
