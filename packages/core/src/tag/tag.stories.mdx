# Tag

Tags show concise metadata in a compact format. Tags are visually styled to differentiate them from buttons.

## Installation

To use the tag component import the component in your JavaScript.

```typescript
import '@clr/core/tag/register.js';
```

```html
<cds-tag status="info">Info</cds-tag>
```
