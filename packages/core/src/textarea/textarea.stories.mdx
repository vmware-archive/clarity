# Textarea

Use Textareas for long-form text input or text input that could be longer than a single sentence.

## Installation

To use the textarea component import the component in your JavaScript.

```typescript
import '@cds/core/textarea/register.js';
```

```html
<cds-textarea>
  <label>textarea</label>
  <textarea></textarea>
  <cds-control-message>message text</cds-control-message>
</cds-textarea>
```
