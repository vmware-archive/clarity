To used on angular: "<clr-search></clr-search>"

Proprietes: 
      [placeholder] - String
[(search)] - String
(searchChange) - String <$event>
[autoCompleateData] - String[]

Simple Search


          <clr-search></clr-search>

Simple Search whit placeholder


          <clr-search [placeholder]="'Search placeholder'"></clr-search>

Simple Search and 1 way binding data whit placeholder

    searchInit = "{{searchInit}}"

          <clr-search [placeholder]="'Search'" [search]="searchInit"></clr-search>

Simple Search and 2 way binding whit placeholder

Separeted events
searchDataS = "{{searchDataS}}"

          <clr-search
          [placeholder]="'Search'"
          [search]="searchDataS"
          (searchChange)="searchDataS = $event"></clr-search>
Tougeder events
searchDataT = "{{searchDataT}}"

          <clr-search [placeholder]="'Search'" [(search)]="searchDataT"></clr-search>

Search whit Autocomplete data

Suports regular expresion 


          <clr-search [placeholder]="'Search'"
      [autoCompleateData]="['This is just an array','abc','def','ght','aoit', 'asdddsaaasdasdasd']">
            </clr-search>
