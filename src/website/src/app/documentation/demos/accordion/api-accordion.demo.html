Summary of Options

<clr-accordion> component

Input / Output Values Default Effect
[clrAccordionMultiPanel]
Type: Boolean
Default: false
 Boolean false Override default behavior to allow multiple panels open at one time.

<clr-accordion-panel> component

Input / Output Values Default Effect
[(clrAccordionPanelOpen)]
Type: Boolean
Default: false
 Boolean false Set and listen for open changes to a given panel.
[clrAccordionPanelDisabled]
Type: Boolean
Default: false
 Boolean false Disable a given panel from being opened or closed.

Input/Output Values Default Effect
