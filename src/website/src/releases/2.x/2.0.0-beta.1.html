Highlights

  • To get this update, run ng update @clr/angular@next. The @next tag is necessary only while this is in prerelease, afterwards it won't be necessary. You can use this with any Angular CLI project, even if you've already started to work on it. You may need to run ng update @angular/core @angular/cli first to update to Angular 8.

    Support for Angular 8-beta.5 (#3146)

    This release brings Angular 8-beta.5 support for Clarity, and this Clarity branch will be supported to work with Angular 8 while Angular is supported.

    Simplifying color variables and removing luminance and contrast maps(#3131)

    We've cleaned up a lot of variables and simplified the CSS theming to make it easier to work with. Custom themes may need to review their themes for any referenced maps and some new variables were added to capture some missing configurations.
