Highlights

  • Custom Triggers for Signpost

    You can now declare custom triggers for a Signpost. To use it, all you have to do is put the clrSignpostTrigger in the top-most container for your custom trigger.

    
<clr-signpost [clrSignpostPosition]="'top-middle'">
    <clr-icon shape="avatar" clrSignpostTrigger></clr-icon>
    <clr-signpost-content *clrIfOpen>
        The pokemon is strong.
    </clr-signpost-content>
</clr-signpost>

Bug Fixes

  • Fixed a height/style issue when Signpost is used in a DatagridCell
