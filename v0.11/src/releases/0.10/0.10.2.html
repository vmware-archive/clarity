Highlights

  • New expanded section for Clarity Icons

    The 0.10.2 release features a new expanded section for Clarity Icons at https://vmware.github.io/clarity/icons. The new section offers many new features and functionalities such as searching icons, viewing an icon in detail with its variants as well as its code, downloading a specific variant of an icon, etc, Moreover, users will find the most up to date Clarity Icons documentation with more Plunkr examples.

  • More icons for Clarity Icons

    With 0.10.2, Clarity has introduced new icons such as cloud-network, cloud-scale, and cloud-traffic.

Bug Fixes

  • Fixed a bug with dropdown menus not closing on item click
  • Fixed an issue with CSS Tooltips. They now remain hidden when the tooltip content is hidden.
  • Fixed an issue that would crash the datagrid when filtering down to 0 elements if single-row actions were present.

Other Changes

  • Updraged Clarity Angular dependencies to use Angular 4.3 or above
  • Removed Dompurify from Clarity Icons
