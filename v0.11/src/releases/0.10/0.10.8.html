Highlights

  • New icons added to Clarity Icons

    Several new icons were added to the Clarity Icons library including: shrink/minimize, magic wand, DNA helix, groups of files, a paperclip icons, and more (#1330)

  • Clarity build is using webpack

    We re-wrote all our scripts to rely on webpack and npm scripts. Read the updated BUILD.md file for more info on how to build and run Clarity locally. (#1509)

    To follow a common webpack convention, the clarity-angular umd file will be moved into the bundles folder after 0.11.0 is released. If you are using SystemJS, you will need to update the main file for clarity-angular when upgrading to 0.11 because the present location has been deprecated.

Bug Fixes

  • Fixed invalid markup for several icons in the technology stack (#1501)
  • Fixed issue for wizard buttons that had NgIf (#1439)
  • Fixed a popover clipping issue in Trees. (#1523)
