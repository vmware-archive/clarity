Highlights

  • The datagrid pagination component can now directly project the pagination description next to the pagination list. This enhancement simplifies the API and allows for easier conditional pagination. (#1854).

    Original API

    
<clr-dg-footer>
    {{pagination.firstItem + 1}} - {{pagination.lastItem + 1}}
    of {{pagination.totalItems}} users
    <clr-dg-pagination #pagination [clrDgPageSize]="10">
    </clr-dg-pagination>
</clr-dg-footer>

    New API

    
<clr-dg-footer>
    <clr-dg-pagination #pagination [clrDgPageSize]="10">
        {{pagination.firstItem + 1}} - {{pagination.lastItem + 1}}
        of {{pagination.totalItems}} users
    </clr-dg-pagination>
</clr-dg-footer>

Bug Fixes

  • Fixed an issue with Datagrid Pagination. Datagrid Pagination now resets the page size when the pagination is disabled. (#1854).
