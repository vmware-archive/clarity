Highlights

  • Clarity icons dynamically project their shape changes now

    All active Clarity icons would observe their shape changes in the API and reflect those changes in the DOM. In other words, this update will allow active icons to project their most recent shape changes, especially in the cases when shape templates change due to asynchronous events.
