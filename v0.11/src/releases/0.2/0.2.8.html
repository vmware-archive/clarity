Highlights

  • Compact styling for forms

    Extends forms with a class (form.compact) that decreases space between rows on the form.

  • Add code block highlighting to UI/NG components

    Adds a code block highlighting UI and Angular component (clr-code-highlight) to Clarity.

  • Compact, non-bordered, and vertical HTML table styles

    0.2.8 introduced several variants of the HTML table styles in Clarity. These include a compact style (.table-compact) decreases the height of table rows from 36px to 24px, a non-bordered style (.table-noborder) removes the border around HTML tables and in between rows (except for below the header), and a vertical style (.table-vertical) that displays an HTML table with the same orientation as the key-value layout of a Stack View. While not documented on the website yet, examples are available on the Clarity demos in the git repo.
