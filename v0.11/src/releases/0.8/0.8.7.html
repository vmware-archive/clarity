Highlights

  • Single row action in datagrid

    Ability to add clickable action overflow menu to each row, in both selectable or non-selectable datagrids.

  • Batch action menu

    Option to display batch action options on top of the datagrid for selected rows.

  • New icons for Clarity Icons

    With 0.8.7, Clarity has introduced new icons such as clipboard, firewall, list, network, redo, undo, etc.

Other Changes

  • Added keyboard support for accessing Tooltips
  • Added a utility class called .u-main-container which can be used when the .subnav is optional on some pages.
