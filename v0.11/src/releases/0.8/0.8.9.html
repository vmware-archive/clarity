Highlights

  • Datagrid: Two-way binding and default values on filtered state

    Added two way binding to filter values for both <clr-dg-column> and <dclr-dg-string-filter>. They now offer the [(clrFilterValue)] property to bind to.

  • Datagrid: Two-way binding on the sorted state of a column

    <clr-dg-column> now offers a [(clrDgSorted)] two-way binding to dynamically force a column to be sorted or not, typically on initialization. It also lets consumers know at all time if a column is sorted or not.

  • Datagrid: Single selection

    Added an option for single selection mode in datagrid. Use [(clrDgSingleSelected)] to use this feature.

Other Changes

  • Moved component styles for form fields from forms.clarity.scss into their own separate files.

Bug Fixes

  • Updated datagrid component to support AOT.
