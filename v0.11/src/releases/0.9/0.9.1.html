Highlights

  • Datagrid column resizing
    Column resizing is enabled in Datagrid by default now. Users can hover over a separator line that divides adjacent column headers and drag it to resize the column before the dragged separator. It's also possible to bind through the (clrDgColumnResize) directive on clr-dg-column to listen to the resizing event and receive its emitted value which will be the new width size of the column.
  • Flip Clarity Icons
    With 0.9.1, Clarity Icons has introduced an ability to flip icons horizontally and vertically. Usage examples: 
    
<clr-icon shape="floppy" flip="horizontal"></clr-icon>
<clr-icon shape="floppy" flip="vertical"></clr-icon>
  • Sketch Template Updated
    The Clarity sketch template has been updated with the latest changes in 0.9.0.
  • New Clarity wizard documentation
    As the 0.9.0 Clarity wizard moves closer and closer to final, we have added extensive documentation on its functionality and the different ways we have seen it used. Check out the new wizards documentation!

Other Changes

  • Datagrid columns are resizable by dragging.
  • Action bar has been enabled in non-selectable Datagrids.
  • Clarity Icons has added new standard size icons for error, warning, success, and info icons.

Bug Fixes

  • Fixed a misalignment issue in Datagrid columns with dynamically loaded *clrDgItems.
  • Fixed an issue with Checkbox and Radio label alignment.
  • Fixed an issue with the content wrapping in the Datagrid footer.
  • Fixed an issue with Button Groups in cards.
