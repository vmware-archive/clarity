Highlights

  • Craft Library updated!

    The Clarity Craft library has been updated to work with the latest versions of Sketch and Craft!

  • Clarity adds a monospace font

    Clarity has decided on a recommendation for monospaced text and has implemented this recommendation in Clarity UI. Find out more information in our typography documentation.

Bug Fixes

  • An issue in Clarity wizards was fixed where it would fire two onCommit events when clicking a Finish button in a wizard with an altNext workflow.
  • Two incorrect color contrast recommendations were fixed in the Clarity color palette documentation.
