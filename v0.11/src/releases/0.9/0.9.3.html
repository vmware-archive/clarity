Highlights

  • Hideable Datagrid Columns

    Datagrid columns are hideable with the *clrDgHideableColumn directive. Look for it on the Datagrid documentation page.

Other Changes

  • Now allowing numeric values to be used in the ids of wizard pages. This is useful for users wanting to use an index as the last part of their wizard page id.

Bug Fixes

  • Fixed "required" input labels for multiple columns in grid layout.
  • Fixed bug with opening multiple filters in datagrid. This bug fix also introduces a slight change in behavior, where the filters close upon scrolling.
  • Stopped the title hint on the clarity icon svgs from showing up when used in tooltip components.
