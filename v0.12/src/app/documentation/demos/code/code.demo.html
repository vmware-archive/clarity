Code Highlight is deprecated and will be removed in 0.13. You can copy the implementation into your own application to maintain it yourself, or use a different code highlight module.
Code Highlight is a custom directive to highlight code blocks using Prism.

Highlighting languages other than HTML, CSS, and JavaScript requires prism.css and prism.js to be included in the page.

To highlight a code block, add the clr-code-highlight directive to the code tag with the value set to the language-* style classes from PrismJS. Refer to the Prism documentation for the list of supported languages.
